## Usando localmente

#### Necessário

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)

#### Comandos

- `git clone git@github.com:murillo94/investments-chart.git`
- `yarn install`
- `yarn dev`

Abra um navegador com a url `http://localhost:3000`
